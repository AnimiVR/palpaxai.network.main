# Services Images

Thư mục này chứa ảnh cho phần "My Services" trong dashboard.

## Cấu trúc

- `service-1.png` - Ảnh cho service đầu tiên (AI Chatbot Service)
- `service-2.png` - Ảnh cho service thứ hai (Content Generation API)  
- `service-3.png` - Ảnh cho service thứ ba (Data Analysis Service)
- ... (thêm ảnh cho các services khác)

## Định dạng khuyến nghị

- **Kích thước:** 800x600px hoặc 1200x900px (tỷ lệ 4:3)
- **Định dạng:** PNG, JPG, hoặc WebP
- **Dung lượng:** Tối ưu để load nhanh (< 500KB)

## Cách sử dụng

Sau khi copy ảnh vào thư mục này, cập nhật đường dẫn trong `ServicesContent.tsx`:

```typescript
image: "/images/services/your-image-name.png"
```
